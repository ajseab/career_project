Intro

Using MATLAB and Python in the Robotics and Electronics lab last year first sparked my interest in programming. Designing and building websites is fun and it allows me to be creative.

What a Web Developer Does

Web developers write code to design, create, and modify websites.

  • - Meet with management to discuss the needs and design of a website

  • - Create and test applications for a website

  • - Write code for a websites, using HTML

  • - Work with other team members

  • - Work with designers to determine the design and layout of the website

  • - Monitor website traffic

Schedule and Work Enviroment

The employers of web developers in order from largest to smallest are: computer systems design and related services, educational services, religious, and other information services. Most web developers work full time.
